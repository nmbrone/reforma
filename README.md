# Refova

A Higher Order Component for form validations in React apps.
Highly inspired by awesome [Formik](https://github.com/jaredpalmer/formik)

### Installing
With npm:
```bash
npm install refova --save
```
or with [Yarn](https://yarnpkg.com):
```bash
yarn add refova
```
